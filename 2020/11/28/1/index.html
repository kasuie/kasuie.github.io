第一篇，也不知道写啥好，总之还是费了很久时间的（~~大概摸了一个月~~）。

使用技术；

后端

  • 核心框架：SpringBoot
  • 项目构建：jdk1.8、Maven 3
  • 持久层：Mybatis
  • 模板引擎：Thymeleaf
  • 其他：Servlet

前端

  • html，css，javascript
  • UI框架：Semantic UI
  • JS框架：JQuery
  • 其他：Ajax

数据库

  • MYSQL 5.7

因为第一次写，参考了Github上两个开源的博客源码。其实主要是为了前端兼容问题，不太擅长做移动端兼容。然后发现页面不太符合我的审美，所以就极大程度的魔改了。因此，到目前为止都还有很多bug和页面的不美观，暂时也不想管了，等后面摸吧。(这里推一下以前练习做的网站：导航，安利墙，github page做的，所以国内访问肯定不快。)
